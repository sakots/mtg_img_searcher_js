# mtg_img_searcher_js

## これは何

[mtg_img_searcher](https://github.com/sakots/mtg_img_searcher)をdiscord.jsで書き直そうとするヤツです。

## 準備

node.js使います。

```node
npm i discord.js dotenv puppeteer
```

.env.exampleにBotのトークンIDを貼り付けて、.envに改名してください。

あとはDiscordBotの動かし方参照。
